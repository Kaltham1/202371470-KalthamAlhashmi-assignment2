# Technical Documentation

## Project Structure
- index.html: Main HTML structure  
- style.css: Styling and layout  
- script.js: Interactivity (toggle + form validation)  
- assets/: Images and CV file  

## Features

### Dynamic Content
- Implemented a "Read More / Read Less" button  
- Allows users to expand and collapse additional content  

### Data Handling
- JavaScript is used to validate user input in the contact form  
- Validation includes:
  - Empty fields check  
  - Email format validation  
  - Phone number validation  

### Error Handling and User Feedback
- Displays error messages for invalid or empty inputs  
- Shows a success message when the form is submitted correctly  

### UI/UX Enhancements
- Smooth scrolling navigation  
- Hover effects on buttons and elements  
- Image animation using CSS keyframes  

## Known Limitations
- No backend integration for form submission  
- Data is not stored permanently
