# AI Usage Report

## Tools Used
- ChatGPT

## How AI Was Used
AI was used to support development in multiple parts of this project:

- Helping implement form validation logic in JavaScript  
- Designing user feedback messages (error and success messages)  
- Fixing CSS layout issues and improving responsiveness  
- Improving UI/UX structure and interaction behavior  
- Assisting in debugging and refining the code  

## Benefits
- Faster problem solving and debugging  
- Better understanding of JavaScript validation logic  
- Improved user experience design  
- Cleaner and more organized code  

## Challenges
- Some AI-generated solutions needed adjustments to fit the existing code  
- Required manual testing to ensure correct behavior  

## Responsible Use
All AI-generated suggestions were reviewed, modified, and tested before being used.  
I ensured full understanding of the implemented logic and did not copy code blindly.
