# Personal Portfolio – Assignment 2

This project is an extension of Assignment 1.
The goal was to enhance the portfolio by adding interactivity, data handling, and better user experience using JavaScript.

## How to Run
Open `index.html` in your browser or use Live Server in VS Code.

## Features
- Dynamic "Read More / Read Less" toggle in the About Me section
- Contact form with input validation
- User feedback messages (error and success messages)
- Responsive design
- Animations and hover effects

## AI Usage (Summary)
AI tools (ChatGPT) were used to:
- Help implement form validation logic
- Improve user feedback messages
- Fix UI/UX issues and styling
More details are available in docs/ai-usage-report.md.
